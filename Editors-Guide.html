Creating Pages

Page creation is available to all registered users. When creating a new page you should use a template from the templates directory found under this editors guide page directory, if a template doesn't already exist it's best to template from an already existing page of the same type after you have chosen a directory and name an option for picking a template will be presented, select that if the page you're making should be based off another page. For example, when adding a new planet to the wiki, use the planet template or if that template didn't exist use the Alderaan page as a template and change all the information from that page.

Editing Pages

Page editing is available to all registered users. Editing pages work the same as creating them, note all changes and edits are monitored, recorded, tracked, and can be rolled back.

Moving/Deleting Pages

Due to limitations in the permissions system, the ability to move or delete pages is restricted, if you think a page should be moved or deleted please contact an administrator.

Style Guides & Templates

Each major directory has its own style for directory pathing and page creation.

Places

The template information for this section can be filled out using information from Wookiepedia as the planet and place information will tend to be the same as the official Star Wars canon. Places should follow the same format as each other so if a template isn't available in Editors-Guide/Templates then you should select an existing page for a similar thing. 

Collecting Information

  • The Star Wars Galaxy Map by x is a great place to get Region and Sector information.
  • Wookiepedia is another great place for this information as well as most of the sidebar, demographic, and descriptive information.

Paths & Naming

Places/<Region>/<Sector>-Sector/<Planet>/<City/Area>/<Point of Interest>

While each field is simply the name of the area (region, planet, etc..) the sector field must be suffixed by -Sector rather than just being the sector name. So rather than just writing Arkanis you would write Arkanis-Sector Here are some example paths:

The planet Alderaan in the Alderaan Sector: Places/Core-Worlds/Alderaan-Sector/Alderaan

The planet Tatooine in the Arkanis Sector: Places/Outer-Rim/Arkanis-Sector/Tatooine

The city of Mos Shuuta on Tatooine: Places/Outer-Rim/Arkanis-Sector/Tatooine/Mos-Shuuta

Teemo's Palace in Mos Shuuta on Tatooine: Places/Outer-Rim/Arkanis-Sector/Tatooine/Mos-Shuuta/Teemos-Palace

For the Arkanis Sector: Places/Outer-Rim/Arkanis-Sector

 

Characters

As character information is almost entirely related to the campaign itself the only real source is from the unfolding story. Most information on a character's page is player provided and from the player's perspective.

Paths & Naming

Characters/<name>

The character B'ura Ban: /Characters/Bura-Ban

 

Vehicles

The template information for this section can be filled out using information from Wookiepedia as vehicles and vehicle information will tend to be the same as the official Star Wars canon. Vehicles should follow the same format as each other so if a template isn't available in Editors-Guide/Templates then you should select an existing page for a similar thing. Most vehicles will have information related to the campaign itself and is player/GM provided.

Named unique vehicles should be tagged with the tag named-vehicle

Paths & Naming

Vehicles/<type>/<name>

The Krayt Fang: /Vehicles/Starships/Krayt-Fang

A generic YT-1300 Freighter: /Vehicles/Starships/YT-1300

A generic Speeder Bike: /Vehicles/Land/Speeder-Bike
